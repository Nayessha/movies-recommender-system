# Movie Recommender System

A content-based movie recommendation system developed using Python, Streamlit, and machine learning techniques.

## Features

* Recommends similar movies based on content similarity
* Fetches real-time movie posters using the TMDB API
* Interactive Streamlit user interface
* Fast recommendation retrieval using a precomputed similarity matrix

## Tech Stack

* Python
* Pandas
* NumPy
* Streamlit
* Requests
* TMDB API

## How It Works

The system uses a precomputed similarity matrix generated from movie metadata. When a user selects a movie, the application identifies the most similar movies and displays recommendations along with their posters.
